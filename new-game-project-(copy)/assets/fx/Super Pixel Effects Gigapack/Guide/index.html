Introduction & Table of Contents


Welcome! This document offers a visual overview of each category of animations found within Super Pixel Effects Gigapack. The goal of this document is to make it easy to browse the effects and view all of the animations in one place.
Each effect comes with a large and a small version, which are displayed as GIFs here. All effects are displayed at 300% zoom and in the color theme I originally drew them in.

Quick Tips:
  • Every animation is intended to be played at 15 FPS (66.7 ms/frame).
  • You can rotate or flip most effects for more variety in your game!

Category: Explosions

Animations which depict explosions, usually involving a fireball and smoke puff together. May include other elements such as lightning.
Explosions have the following subtypes:
  • Epic explosion: Explosions which are grander in scale / more detailed.
  • Stylized explosion: Explosions which have their own particular flair.
  • Symmetrical explosion: Explosions which have vertical, horizontal, diagonal, or rotational symmetry. (Tend to be more videogame-like.)

Category: Impacts

Animations which depict a burst of particles, as if from one object hitting another. They usually combine an expanding ring and several linear particles to convey the force of the impact. Very versatile animations which are great for fighting games, damage effects, and more.
Impacts have the following subtypes:
  • Directional impact: An impact which implies a particular direction, as if bouncing off of or repelled by the object being impacted.Rotate the effect to change the direction.
  • Symmetrical impact: Directionless impacts which are typically circular and have vertical, horizontal, diagonal, or rotational symmetry.
  • Toon impact : An impact which is cartoonlike, usually featuring exaggerated features such as stars. (Can be directional or symmetrical, but is always labeled as "toon.")

Category: Lightning

Animations which depict lightning as their primary feature.
Lightning has the following subtypes:
  • Lightning strike: One or more lightning bolts which follows a specific path/direction. These animations usually have an impact animation, and some have a second version without an impact.
    • (A) Lightning strikes marked with the "A" suffix have an impact.
    • (B) Lightning strikes marked with the "B" suffix are the same as A, without the impact.
  • Lightning burst: Animations which primarily depict a shocking burst of electricity. Great for powerups, damage effects, or as sci-fi explosions.

Category: Symbols

Animations which depict symbols such as checkmarks, X's, and so on.
These animations blur the line between effect and UI element, and can be used as either one.
Some symbols (such as Symbols/symbol_success_001, for example) may have a distinct start, loop, and finish animation. These animations are not separated by default, but can be separated manually as your game or project requires. For example, you could have a checkmark appear (start), shine and sparkle (loop) for some duration, then disappear (finish).
