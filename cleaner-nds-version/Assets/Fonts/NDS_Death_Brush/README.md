# NDS Brush Display Font

`Knewave-Regular.ttf` is the closest complete, game-ready match selected for
the hand-painted lettering in `NDS_logo16bit.png`.

## Coverage

- Uppercase and lowercase Latin alphabet
- Numbers
- Common punctuation and symbols
- Extended Latin characters

## Recommended Godot use

- Use for titles, mission names, boss cards, and short labels.
- Keep body copy in a simpler pixel font.
- Use white lettering with a 3-5 px black outline at large sizes.
- Slightly rotate individual title labels or place them on an arc when matching
  the original logo composition.
- For the closest logo treatment, use uppercase text at 64 px or larger.

The font is distributed under the SIL Open Font License 1.1. See `OFL.txt`.
The original logo lettering remains a custom composition; a font cannot
automatically reproduce its arched baseline and hand-positioned strokes.
